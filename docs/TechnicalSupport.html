技术支持

  • 请访问 QZhi Studio 的 Github 主页 https://github.com/QZhiStudio/ 检查您是否拥有最新版本的 QZhi HTML Editor 软件。
  • 请查阅常用问题解答，是否您的问题已经有答案。
  • 技术支持是免费的。您可以在 Github 上提交 issue。
  • 为了提供及时准确的服务，报告错误时请提供以下信息：
    1. 问题是否可重现？如何重现？
    2. 您使用的操作系统版本（Windows 2000, Windows XP 等）。
    3. 您使用的 QZhi HTML Editor 版本。
    4. 如果有错误提示对话框，请提供完整的错误信息，最好包含截图。
